# YAC/VAYS Documentation

Visit https://yac-vays.github.io for the documentation.

## Development

To run locally:

```bash
sudo apt install ruby-dev build-essential libffi-dev libssl-dev zlib1g-dev libxml2-dev libxslt1-dev
bundle install
bundle exec jekyll serve --livereload
```
